## Режимы продукта

Сейчас у ToraSEO есть три пользовательских AI-пути:

| Режим | Что делает |
|---|---|
| **MCP + Instructions** | ToraSEO готовит сканирование, Codex или Claude Desktop вызывает MCP-инструменты, а приложение получает структурированные результаты. |
| **API + AI Chat** | ToraSEO выполняет workflow внутри приложения через настроенного провайдера и модель. |
| **Skill без MCP и APP** | Chat-only fallback, когда instruction package доступен, но desktop app, MCP server или активный запуск недоступны. |

## Правила GitHub-документации

- Английский язык — основной язык документации.
- Русские страницы должны передавать смысл продукта, а не копировать английский текст дословно.
- Скриншоты приложения могут оставаться английскими до подготовки локализованных скриншотов.
- Утверждения про ранжирование, трафик, backlinks, клики, показы или SERP должны опираться на официальный источник данных, когда такая интеграция появится.
- AI-writing probability и AI trace signals — это эвристики для редактуры, а не доказательство авторства.

## Документация релизов

Release notes должны оставаться практичными:

- что изменилось
- какие assets входят в релиз
- как установить или обновиться
- что проверено
- известные ограничения
- ссылки на нужную документацию

Исторические черновики релизов находятся в [docs/releases](releases/).
