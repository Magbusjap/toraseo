# FAQ - ToraSEO

**Язык:** [English](FAQ.md) | Русский

Короткие ответы про режимы ToraSEO, инструменты аналитики, ИИ-провайдеров, отчеты, экспорт и приватность.

## Режимы

### Что такое MCP + Instructions?

`MCP + Instructions` связывает ToraSEO с Codex или Claude Desktop. ToraSEO готовит контекст анализа, внешний AI-клиент вызывает выбранные MCP-инструменты, а приложение получает структурированные результаты.

Используйте этот режим, когда нужен ответ во внешнем чате по tool evidence и структурированный отчет в ToraSEO.

### Что такое API + AI Chat?

`API + AI Chat` оставляет workflow внутри ToraSEO. Приложение собирает факты сканирования, отправляет выбранной модели только нужные evidence и формирует отчет в приложении.

Используйте этот режим, когда не хочется вставлять bridge-команды и нужно продолжать работу во встроенном чате.

### Что такое Skill без MCP и APP?

Это chat-only fallback. Он полезен, когда instruction package установлен, но ToraSEO Desktop App, MCP server или активный запуск недоступны.

ИИ отвечает в чате по вставленным или видимым фактам. Desktop-отчет в этом режиме не обновляется.

## Инструменты аналитики

### Почему в сайдбаре написано «Дополнительные проверки»?

Базовые проверки входят в пакет выбранного анализа. В сайдбаре показываются дополнительные проверки, которыми можно расширить или сузить отчет.

### Почему сравнение сайтов не показывает три полных аудита рядом?

Сравнение сайтов — это competitive dashboard, а не три отдельных отчета. Он должен отвечать:

- кто сильнее
- почему
- где разрыв
- что исправить первым

Поэтому ToraSEO использует компактные карточки сайтов, сравнительные метрики, heatmap, победителей и практические выводы.

## ИИ-провайдеры

### Какие ИИ-провайдеры поддерживаются?

OpenRouter обозначен как международный роутер моделей. RouterAI обозначен как российский OpenAI-compatible роутер с оплатой в рублях.

Оба настраиваются в Settings. Добавьте ключ провайдера, сохраните один или несколько ID моделей и выберите одну модель по умолчанию для приложения.

### Для RouterAI нужно вставлять функцию в настройки?

Для обычного чата и анализа не нужно. RouterAI предоставляет OpenAI-compatible API endpoint, поэтому ToraSEO может использовать тот же adapter chat completions с базовым адресом RouterAI.

Плагины RouterAI, например web search, можно позже добавить как опции провайдера. Их не нужно вставлять большим кодом в поле ID модели.

## Отчеты и экспорт

### Где указана версия анализа?

В отчетах отдельно показываются версия приложения и версия анализа. Версия анализа показывает, по каким пользовательским правилам был собран отчет.

### Можно ли экспортировать отчет?

PDF-экспорт доступен для отчетов. Для сравнения сайтов лучше использовать горизонтальную ориентацию, потому что широкий сравнительный dashboard читается лучше узкого вертикального листа.

Экспорт в презентацию тоже хорошо подходит для будущих competitive comparison reports.

## Приватность

### Что отправляется в интернет?

ToraSEO отправляет запросы к тем URL, которые вы сами анализируете. В `API + AI Chat` выбранный провайдер также получает факты сканирования и промпт, нужный для формирования отчета.

Сохраненные API-ключи хранятся через защищенные настройки провайдера и не показываются обратно открытым текстом.

### ToraSEO соблюдает robots.txt?

Да. Поведение сканера описано в [CRAWLING_POLICY.md](../CRAWLING_POLICY.md).

### Какие части еще в разработке?

`Дизайн и контент по URL` и `Анализ изображения` сейчас отмечены как находящиеся в разработке. Tora Rank — будущий вектор, а не готовая публичная scoring-система в этом релизе.
